ℹ️ Para que serve isso?

Se você precisar adicionar dados de vários meses ou várias unidades de uma vez, em vez de preencher o formulário acima entrada por entrada, pode exportar um arquivo JSON (botão em Configurações), editá-lo no Excel ou Google Sheets, e importar tudo de uma vez aqui.

O formato esperado é uma lista de entradas, cada uma com os campos:
rede, unidade, mes, tipo, invest, leads, alcance, impressoes, cliques.

Na prática, você provavelmente não precisará usar isso — o formulário acima é suficiente para atualizações mensais normais.
